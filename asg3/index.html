Creative Brief

Problem

There is no to-do list website that is fully intuitive to a wide, diverse range of user tasks including nuanced deadlines and tasks within tasks.

Audience

Anyone could benefit from keeping a to-do list, but my target would be college students like myself who need a to-do list site that allows them to sort priorities, sub-tasks, and multi-day projects.

Competition

There are many to-do list websites out there already. The creation of this project is pushed by the fact that none of them have all the features necessary to work for what I and many other people I know need. The goal of this project is to take something that has already been created and make it better.

Key Messages

My goal is to make an intuitive to-do list system that allows for more than just putting items on a list for a single day. The appeal of this site should be the variety of task options and deadlines it offers.

Call to Action

Getting tasks organized can be the difference between success and failure, both at college and in the workplace. My goal is to create a way to organize tasks that allows for more than just "do this today" or "do this by x date".

How to Measure Success

User traffic is an important metric, but especially traffic from returning users - it means a lot if people like it enough to come back. Additionally, users can be surveyed upon making their first list and after using the site for a while to determine if the site has really helped them to organize their tasks more efficiently.
